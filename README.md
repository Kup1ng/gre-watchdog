# gre-watchdog

نصب Agent روی خارج: clone → config → install_agent.sh

نصب Coordinator روی ایران: clone → config → install_coordinator.sh

تنظیم shared_secret یکسان

محدود کردن allow_cidrs در Agent فقط به IP ایران

باز کردن پورت 7801 بین ایران→خارج

پنل روی 8000

فعال کردن nginx+tls (اختیاری ولی توصیه‌شده)

توضیح منطق ping و reset و confirm_bad_rounds و loss_ok_percent
